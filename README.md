# Construction Corner

### India's #1 Construction Network App

---

## About

Construction Corner is a super-app for India's construction industry — a single platform that connects **contractors, professionals, workers, suppliers, and clients** to discover opportunities, hire talent, source materials, and grow their business.

---

---

## Key Features

| Feature | Description |
|---|---|
| **4 Profile Types** | Worker, Professional, Business, Supplier |
| **Verified Work Done Badge** | Immutable, on-chain-style work history |
| **Material Marketplace** | Buy and sell construction materials |
| **Equipment Rentals** | Rent machinery and tools |
| **Job Board** | Post and find construction jobs |
| **B2B Connect** | Company-to-company deal flow |
| **Course Platform** | Upskilling for construction professionals |
| **Real-time Chat** | In-app messaging for all users |
| **CC Trust Score** | Composite credibility score per user |
